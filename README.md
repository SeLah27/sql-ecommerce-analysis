# sql-ecommerce-analysis
Analyzed e-commerce dataset (100k+ records) using SQL, building complex JOIN queries and extracting business insights such as customer value, revenue trends, and top-performing products.

#Tools Used
SQL (SSMS)
Relational Database (multiple tables with JOIN operations)

#Dataset
Source: Kaggle (Brazilian E-commerce Dataset)
Contains orders, customers, products, and transactions data

#Key Analysis Performed
Total Revenue calculation
Top-selling products identification
Revenue distribution by region
Monthly sales trend analysis
Top customers by spending
Customer Lifetime Value (CLV)
Average Order Value (AOV)

#Key Insights
A small group of customers generates a large portion of revenue
Sales are concentrated in specific regions
Revenue shows variation across months indicating possible seasonality
High-value customers can be targeted for retention strategies

#What I Learned
Writing complex SQL queries using JOIN, GROUP BY, and aggregation
Transforming raw data into business insights
Understanding customer and sales behavior through data
